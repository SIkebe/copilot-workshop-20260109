# CONTRIBUTING

このプロジェクトへのコントリビュートに関心をお寄せいただきありがとうございます。良い体験にするため、次のガイドラインに沿ってご協力ください。

## 行動規範

参加者は互いに敬意を払い、誰もが安心して参加できる環境づくりに努めてください。差別的、攻撃的な言動は控え、建設的なコミュニケーションを心掛けてください。

## 相談・課題報告

- バグ報告や機能提案は Issue を作成してください。
- 期待する挙動や再現手順、使用環境（Node.js のバージョンなど）をできるだけ具体的に記載してください。

## 開発の流れ

1. リポジトリをフォークし、作業用ブランチを作成します。
2. 依存関係をインストールします。
   ```bash
   npm install
   ```
3. 変更内容を実装し、テストを実行して動作を確認します。
   ```bash
   npx jest
   ```
4. 目的が明確になるようにコミットメッセージを付け、プルリクエストを作成します。

## コーディング指針

- 可能な限り既存のコードスタイルに合わせてください。
- テストを追加・更新し、変更点が正しく動作することを確認してください。
- 小さくまとまった変更単位でプルリクエストを作成し、背景や目的を説明してください。

## ライセンス

このリポジトリへのコントリビュートは、リポジトリのライセンスに従うものとします。
