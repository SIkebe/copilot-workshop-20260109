# copilot-workshop-20260109

## プロジェクト概要
- FizzBuzz の簡単な実装と、クラス定義練習用のサンプルコードを含むワークショップ用リポジトリです。

## 前提条件
- Node.js (18 以降を推奨)
- npm

## セットアップ
```bash
npm install
```

## 実行方法
FizzBuzz の出力を確認する:
```bash
node fizzbuzz.js
```

## テスト
Jest を直接実行します:
```bash
npx jest --runInBand
```

> `fizzbuzz.test.js` には学習用の意図的に失敗するテストケースが含まれています。すべてのテストを成功させるには、内容を修正するか不要な失敗テストを削除してください。
